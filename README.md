# RetrofitRxDownload
retrofit2 + rxjava2 下载文件

Step 1. Add the JitPack repository to your build file

  allprojects {<br>
  	repositories {<br>
			...<br>
			maven { url 'https://jitpack.io' }<br>
		}<br>
	}
  
Step 2. Add the dependency

  dependencies {
	        compile 'com.github.cwyrzz:RetrofitRxDownload:1.0.0'
	}
